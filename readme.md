# next-plugin-boilerplate

> Next.js plugin boilerplate

# Installation

```sh
npm install --save next-plugin
```
or

```sh
yarn add next-plugin
```

# Usage

```js
// next.config.js
const withNextPlugin = require('next-plugin')

module.exports = withNextPlugin()
```
